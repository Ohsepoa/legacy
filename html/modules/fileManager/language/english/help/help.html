General Settings

Before using the module, you must do the initial setup.

  • Default upload path
    The standard specifies the folder name to upload. Specified folder, XOOPS installation bus / uploads / directory can be specified below.
    If not specified, XOOPS installation bus / uploads /.
    XOOPS installation bus / uploads / other settings are invalid. XOOPS Installation Bus / myimage / subject is not specified.
  • Thumbnail size
    When you specify a list of thumbnail-size files.
  • Hide System Image
    Do not display images and avatars are using the system and image manager. Recommends hidden. The default is "No".
  • Debugging for uploader
    SWFUpload enable debug screen. Error analysis can help to upload. The default is "No".
  • Support the operation of the folder
    "Yes" to enable the operation of the folder and select. The default is "No".
